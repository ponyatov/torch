# Install

```
$ cd ~ ; git clone -o gh https://github.com/ponyatov/torch
$ cd torch
```
```
$ python3 -m venv .
$ . bin/activate
(venv) $ make install
```

## System requirements

* Debian GNU/Linux 9+
  * `sudo apt install git make python3`
